## Project Overview

**Purse Maison** is an integrated consignment management system with sales forecasting analytics developed for **Purse Maison**, a luxury consignment business specializing in authentic pre-owned luxury items.

The application aims to centralize business operations by providing a digital platform for managing consignment items, inventory records, client assignments, sales transactions, and sales forecasting insights. It helps improve data organization, monitoring of item performance, and decision-making through analytics.

---

## Project Objectives

The Purse Maison Flutter Application aims to:

* Provide a centralized system for managing consignment business records.
* Improve inventory tracking and item status monitoring.
* Organize client and sales associate assignments.
* Manage sales transaction records efficiently.
* Provide sales forecasting analytics to support business decisions.
* Display business insights through an analytics dashboard.

---

## Key Features

### 📈 Main Dashboard

* Display key performance indicators (KPIs).
* Present sales and inventory insights through charts and visualizations.

### 🔐 User Authentication

* Secure user login and account management using Firebase Authentication.
* Role-based access for different system users.

### 📦 Inventory Management

* Add, update, view, and manage consigned luxury items.
* Monitor item details, availability, and inventory status.

### 🤝 Consignment Management

* Manage consignment item records.
* Track item information throughout the consignment lifecycle.

### 👥 Client Assignment Management

* Manage client information and assignments.
* Monitor sales associate responsibilities.

### 📊 Sales Forecasting Analytics

* Provide sales trend analysis and forecasting insights.
* Support business planning using historical sales data.

## 🛠️ Technologies Used

### Frontend

**Flutter**

* Flutter was used as the frontend framework for developing the cross-platform mobile application.
* It provides a responsive user interface and efficient application development.

**Dart**

* Dart is the programming language used to build the application's user interface, logic, and functionality.

### Backend

**Firebase**

* Firebase serves as the backend platform for managing application data and user authentication.

Firebase services implemented:

**Firebase Authentication**

* Handles user authentication and secure login access.

**Cloud Firestore**

* Serves as the cloud database for storing and retrieving application data such as inventory records, client information, sales transactions, and forecasting data.

## Getting Started

### Prerequisites

Before running the application, ensure that you have:

* Flutter SDK installed
* Dart SDK installed
* Android Studio or Visual Studio Code
* Firebase project configuration

### Installation

1. Clone the repository:

```bash
git clone https://github.com/dhzl17/purse-maison-flutter.git
```

2. Navigate to the project folder:

```bash
cd purse-maison-flutter
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the application:

```bash
flutter run
```
